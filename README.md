### Установка 

Необходимо установить все зависимости из requirements.txt

```sh
pip install -r requirements.txt
```
 
 ### Запуск 

Необходимо установить все зависимости из requirements.txt
Перед запуском создайте виртуальное окружение

```sh
pip install virtualenv
virtualenv <env_name>
pytest
```
